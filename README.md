# Mentoria Hub

MVP EdTech-платформы для хакатона **Mentoria — Working MVP Challenge**.

Платформа объединяет две функции: **поиск образовательных возможностей** (конкурсы, олимпиады,
стипендии, стажировки, летние школы) и **асинхронное обучение** через курсы Mentoria.

## ✨ Что внутри

**Для учеников**
- 🏠 Лендинг с ценностным предложением и CTA
- 🔎 Каталог из 12 возможностей с фильтрами (направление, категория, формат, класс, дедлайн) и поиском
- ❤️ Сохранение возможностей в избранное
- 🎓 3 курса с уроками, видео-плейсхолдером, материалами, мини-тестами и прогресс-баром
- 🧭 Онбординг: класс, интересы, предметы, цели
- ✨ Система рекомендаций на основе профиля
- 🤖 AI-ассистент (rule-based, работает офлайн) — подбирает возможности и курсы по запросу
- 📊 Личный кабинет: статистика, избранное, прогресс, календарь дедлайнов, сертификаты
- 🏆 Сертификаты за пройденные курсы

**Для администраторов Mentoria**
- 🛠️ Админ-панель: CRUD возможностей, курсов и уроков (с редактором тестов)
- 📈 Статистика и список пользователей

**Бонусы:** тёмная/светлая тема, мультиязычный интерфейс (RU/EN/KZ), адаптивный дизайн,
поиск, календарь дедлайнов, AI-ассистент, сертификаты.

## 🧱 Стек

| Слой | Технологии |
|------|-----------|
| Backend | Node.js, Express, **Prisma ORM**, **SQLite** (реальная реляционная БД), JWT, bcrypt |
| Frontend | React, Vite, React Router, Tailwind CSS |

Данные хранятся в настоящей базе (SQLite-файл `server/prisma/dev.db`), а не в localStorage.

## 🚀 Запуск

Из папки `mentoria-hub`:

```bash
# 1. Установка зависимостей + генерация БД + наполнение демо-данными + сборка фронта
npm run setup

# 2. Запуск (один порт, отдаёт фронт + API)
npm start
# → http://localhost:4000
```

### Режим разработки (hot reload)

```bash
npm run dev
# фронт: http://localhost:5173  (проксирует /api на :4000)
# API:   http://localhost:4000
```

## 🗺️ Пользовательский путь (для демо)

1. Регистрация → онбординг (класс, интересы, цели)
2. Каталог → фильтр по направлению → сохранить хакатон и летнюю программу
3. Открыть курс «Английский для академического успеха» → пройти урок + мини-тест
4. Личный кабинет → прогресс, дедлайны, рекомендации, сертификат
5. Войти как админ → добавить новую олимпиаду / курс через админ-панель

## 📡 API (основное)

```
POST /api/auth/register | /login        GET /api/auth/me
PUT  /api/users/me                       (онбординг/профиль)
GET  /api/opportunities[?q,category,direction,format,grade,sort]
GET  /api/opportunities/:id              POST|PUT|DELETE (admin)
GET  /api/saved   POST|DELETE /api/saved/:id
GET  /api/courses /:id   POST /api/courses/:id/enroll
POST /api/lessons/:id/complete
GET  /api/dashboard                       (агрегат кабинета)
POST /api/assistant                       (AI-подбор)
GET  /api/admin/stats | /admin/users
```
