# 🎓 Mentoria Hub

### Образовательные возможности и асинхронные курсы — на одной платформе

Рабочий **full-stack MVP** EdTech-платформы для хакатона **Mentoria — Working MVP Challenge**.
Ученики находят конкурсы, олимпиады, стипендии и стажировки, и учатся в своём темпе
через курсы Mentoria — без привязки к живым занятиям.

## 🧩 Проблема

Сегодня Mentoria работает через Telegram и живые онлайн-занятия. По мере роста этого
становится недостаточно:

- не каждый ученик может посещать живые занятия (школа, экзамены, часовые пояса, расписание);
- информация о конкурсах, олимпиадах, стипендиях и стажировках **разбросана** по сайтам, каналам и чатам;
- ученику сложно понять, что подходит именно ему — по возрасту, интересам и целям;
- ручные обновления в Telegram **не масштабируются**.

## 💡 Решение

**Mentoria Hub** объединяет две функции в одном продукте:

1. **Каталог возможностей** — единое место для конкурсов, олимпиад, стипендий, стажировок и летних школ с фильтрами и рекомендациями.
2. **Асинхронные курсы** — структурированное обучение с уроками, тестами и отслеживанием прогресса.

Плюс **админ-панель**, через которую команда Mentoria добавляет контент **без пересборки сайта** — это и есть ключ к масштабированию.

## ✨ Возможности

### Для учеников
- 🏠 **Лендинг** с понятным ценностным предложением и призывами к действию
- 🔎 **Каталог из 12 возможностей** с фильтрами (направление, категория, формат, класс, дедлайн) и поиском
- ❤️ **Избранное** — сохранение возможностей
- 🎓 **3 курса** с уроками, видео-плейсхолдером, материалами, мини-тестами и прогресс-баром
- 🧭 **Онбординг** — класс, интересы, предметы, цели
- ✨ **Рекомендации** на основе профиля (теговая логика)
- 🤖 **AI-ассистент** — подбирает возможности и курсы по запросу (rule-based, работает офлайн, понимает RU и EN)
- 📊 **Личный кабинет** — статистика, прогресс, календарь дедлайнов, сертификаты
- 🏆 **Сертификаты** за полностью пройденные курсы

### Для администраторов Mentoria
- 🛠️ **CRUD** возможностей, курсов и уроков (с редактором тестов)
- 📈 **Статистика** платформы и список пользователей

### Бонусы
🌙 Тёмная/светлая тема · 🌍 Мультиязычный интерфейс (RU/EN/KZ) · 📱 Адаптивный дизайн ·
🔍 Поиск · 📅 Календарь дедлайнов · ♿ Доступность (a11y по чек-листу UI/UX)

## 🧱 Технологии

| Слой | Технологии |
|------|-----------|
| **Backend** | Node.js, Express, **Prisma ORM**, **SQLite**, JWT, bcrypt |
| **Frontend** | React, Vite, React Router, Tailwind CSS |
| **Дизайн** | Единый набор SVG-иконок, шрифты Lexend + Inter, тёмная тема |

> Данные хранятся в **настоящей реляционной БД** (SQLite-файл через Prisma), а не в localStorage.
> Бэкенд на одном порту отдаёт и API, и собранный фронтенд — удобно для демо.

## 🚀 Быстрый старт

```bash
git clone https://github.com/qprtfck-arch/sat.git
cd sat

# Установка зависимостей + генерация БД + демо-данные + сборка фронта
npm run setup

# Запуск (один порт: фронт + API)
npm start

```

### Режим разработки (hot reload)

```bash
npm run dev
# фронт: http://localhost:5173 (проксирует /api на :4000)
# API:   http://localhost:4000
```

> Требуется Node.js 18+. Перед стартом скопируйте `server/.env.example` → `server/.env`
> (или это сделает `npm run setup`).

## 🗺️ Пользовательский путь (демо-сценарий)

1. Регистрация → онбординг (класс, интересы, цели)
2. Каталог → фильтр по направлению → сохранить хакатон и летнюю программу
3. Открыть курс «Английский для академического успеха» → пройти урок + мини-тест
4. Личный кабинет → прогресс, дедлайны, рекомендации, сертификат
5. Войти как админ → добавить новую олимпиаду / курс через админ-панель

```
mentoria-hub/
├── server/                 # Backend (Express + Prisma + SQLite)
│   ├── prisma/
│   │   ├── schema.prisma   # модель данных
│   │   └── seed.js         # демо-данные (аккаунты, 12 возможностей, 3 курса)
│   └── src/
│       ├── index.js        # entry: API + отдача фронта
│       ├── lib/            # prisma, auth (JWT), recommend (рекомендации)
│       └── routes/         # auth, users, opportunities, saved, courses,
│                           # lessons, dashboard, admin, assistant
└── web/                    # Frontend (React + Vite + Tailwind)
    └── src/
        ├── pages/          # Home, Opportunities, Courses, Dashboard, Admin, ...
        ├── components/     # Layout, Icon, OpportunityCard, CourseCard, ...
        └── lib/            # api, store (контекст), i18n, ui-хелперы
```

## 🧬 Модель данных

`User` · `Opportunity` · `SavedOpportunity` · `Course` · `Lesson` · `Enrollment` · `LessonProgress`

Связи: пользователь сохраняет возможности, записывается на курсы и накапливает прогресс по урокам;
курс содержит уроки; у урока — тест (JSON). Сертификат вычисляется, когда все уроки курса пройдены.

## 📡 API (основное)

```
POST /api/auth/register | /login          GET  /api/auth/me
PUT  /api/users/me                          # онбординг / профиль
GET  /api/opportunities[?q,category,direction,format,grade,sort]
GET  /api/opportunities/:id                 POST|PUT|DELETE (admin)
GET  /api/saved   POST|DELETE /api/saved/:id
GET  /api/courses /:id   POST /api/courses/:id/enroll
POST /api/lessons/:id/complete              # прогресс + результат теста
GET  /api/dashboard                         # агрегат кабинета
POST /api/assistant                         # AI-подбор
GET  /api/admin/stats | /admin/users        # CRUD контента (admin)
```

## 🧪 Тестовые данные (mock data)

Что является демо-данными, а не «прод-логикой»:

- **Контент**: 12 возможностей и 3 курса с уроками/тестами заполняются через `seed.js`.
- **Видео уроков** — плейсхолдеры (поле `videoUrl` пустое); поддерживается вставка embed-URL.
- **Ссылки «Подать заявку»** ведут на демо-URL.
- **AI-ассистент** — детерминированная логика на тегах и ключевых словах (без внешнего API), чтобы демо работало офлайн.

Всё остальное — настоящее: регистрация/JWT, сохранение в БД, прогресс, фильтры, CRUD.

## 📈 Как это помогает Mentoria масштабироваться

- Контент добавляется **через админку**, без правок кода и пересборки — больше учеников без роста ручной работы.
- **Асинхронные курсы** снимают зависимость от живых занятий и часовых поясов.
- Единый каталог и рекомендации **повышают вовлечённость и удержание**.
- Профессиональный вид платформы усиливает доверие **партнёров, школ и спонсоров**.
